   - image (varchar 255, nullable) - path gambar
   - meta_data (json, nullable) - data tambahan
   - is_active (boolean, default true)
   - created_at (timestamp)

2. TABLE: news
   -----------
   - id (bigint, primary key, auto increment)
   - title (varchar 255)
   - slug (varchar 255, unique)
   - excerpt (text, nullable) - ringkasan berita
   - content (longtext)
   - image (varchar 255, nullable)
   - is_featured (boolean, default false) - untuk berita utama
   - is_published (boolean, default true)
   - published_at (timestamp, nullable)
   - meta_title (varchar 255, nullable) - SEO
   - meta_description (text, nullable) - SEO
   - created_at (timestamp)
   - updated_at (timestamp)

3. TABLE: galleries
   ----------------
   - id (bigint, primary key, auto increment)
   - title (varchar 255, nullable)
   - image (varchar 255)
   - description (text, nullable)
   - alt_text (varchar 255, nullable) - untuk SEO
   - is_featured (boolean, default false) - untuk carousel
   - sort_order (integer, default 0) - untuk urutan tampil
   - is_active (boolean, default true)
   - created_at (timestamp)
   - updated_at (timestamp)

4. TABLE: food_items
   -----------------
   - id (bigint, primary key, auto increment)
   - name (varchar 255)
   - slug (varchar 255, unique)
   - description (text, nullable)
   - image (varchar 255, nullable)
   - price (decimal 10,2, nullable)
   - category (varchar 100, nullable) - appetizer, main_course, dessert, drink
   - is_featured (boolean, default false) - untuk tampil di home
   - is_available (boolean, default true)
   - sort_order (integer, default 0)
   - created_at (timestamp)
   - updated_at (timestamp)

5. TABLE: contact_messages
   -----------------------
   - id (bigint, primary key, auto increment)
   - subject (varchar 255)
   - name (varchar 255)
   - email (varchar 255)
   - message (longtext)
   - is_read (boolean, default false)
   - replied_at (timestamp, nullable)
   - created_at (timestamp)
   - updated_at (timestamp)

6. TABLE: settings
   ---------------
   - id (bigint, primary key, auto increment)
   - key (varchar 100, unique)
   - value (text, nullable)
   - type (enum: 'text', 'email', 'phone', 'url', 'json')
   - group (varchar 50, nullable) - contact, social, general
   - description (varchar 255, nullable)
   - created_at (timestamp)
   - updated_at (timestamp)

7. TABLE: users (sudah ada)
   --------
   - id (bigint, primary key, auto increment)
   - name (varchar 255)
   - email (varchar 255, unique)
   - email_verified_at (timestamp, nullable)
   - password (varchar 255)
   - role (enum: 'admin', 'editor', default 'editor')
   - is_active (boolean, default true)
   - remember_token (varchar 100, nullable)
   - created_at (timestamp)
   - updated_at (timestamp)

FITUR ADMIN PANEL YANG DIBUTUHKAN:
==================================

1. DASHBOARD
   - Statistik: Total berita, galeri, pesan kontak
   - Berita terbaru
   - Pesan kontak terbaru

2. KELOLA KONTEN
   - Edit konten statis (hero, about, visi, misi)
   - Upload gambar untuk setiap konten

3. KELOLA BERITA
   - CRUD berita
   - Set featured news
   - Upload gambar berita
   - SEO meta tags

4. KELOLA GALERI
   - CRUD galeri
   - Set featured untuk carousel
   - Drag & drop untuk sorting
   - Bulk upload

5. KELOLA MENU MAKANAN
   - CRUD food items
   - Set featured untuk home page
   - Kategori makanan
   - Upload gambar

6. PESAN KONTAK
   - View semua pesan
   - Mark as read
   - Reply pesan (opsional)

7. PENGATURAN
   - Contact information
   - Social media links
   - Google Maps embed
   - General settings

KEUNTUNGAN STRUKTUR INI:
=======================

✅ FLEKSIBEL - Admin bisa edit semua konten tanpa coding
✅ SCALABLE - Mudah tambah konten baru
✅ SEO FRIENDLY - Meta tags, slugs, alt text
✅ USER FRIENDLY - Interface admin yang mudah
✅ MAINTAINABLE - Struktur database yang clean
✅ PERFORMANCE - Optimized queries dengan indexing
✅ SECURITY - Role-based access control
